# 默认闹钟声音目录

## 📁 目录说明

这个目录用于存放默认的闹钟声音文件。

## 🎵 推荐的声音文件

请从免费音效网站下载以下类型的声音文件：

### 基础包（5个声音）

1. **alarm_01_classic_bell.wav** - 经典机械闹钟铃声
   - 风格：传统、熟悉
   - 适用：日常使用

2. **alarm_02_electronic_beep.wav** - 电子哔哔声
   - 风格：现代、简洁
   - 适用：办公、学习

3. **alarm_03_gentle_piano.wav** - 柔和钢琴旋律
   - 风格：温和、舒缓
   - 适用：早晨唤醒

4. **alarm_04_bird_chirping.wav** - 清晨鸟鸣声
   - 风格：自然、清新
   - 适用：周末、假期

5. **alarm_05_emergency_alert.wav** - 紧急警报声
   - 风格：强烈、紧迫
   - 适用：重要事项

## 📥 如何获取声音文件

### 推荐网站（完全免费）

1. **Pixabay** (推荐)
   - 网址：https://pixabay.com/sound-effects/search/alarm/
   - 特点：完全免费，无需署名，可商用
   - 搜索关键词：alarm, bell, notification

2. **Freesound**
   - 网址：https://freesound.org/search/?q=alarm+clock
   - 特点：大量音效，需要注册
   - 搜索关键词：alarm clock, morning alarm

3. **Mixkit**
   - 网址：https://mixkit.co/free-sound-effects/alarm/
   - 特点：高质量音效，可商用
   - 搜索关键词：alarm, notification

## 🔧 文件要求

### 格式
- 支持：WAV, MP3, WMA, M4A
- 推荐：WAV（无损）或 MP3（压缩）

### 质量
- 采样率：44.1 kHz 或 48 kHz
- 比特率：128 kbps 以上（MP3）
- 时长：5-30 秒

### 命名
- 格式：`alarm_[编号]_[描述].wav`
- 示例：`alarm_01_classic_bell.wav`

## 🚀 快速开始

### 方法1：使用下载助手脚本

1. 双击运行 `download_helper.ps1`
2. 选择选项 1 打开 Pixabay
3. 下载喜欢的声音文件
4. 保存到本目录
5. 按照命名规范重命名

### 方法2：手动下载

1. 访问 https://pixabay.com/sound-effects/search/alarm/
2. 搜索并下载 5 个闹钟声音
3. 将文件保存到本目录
4. 重命名为推荐的文件名

## 📝 使用方法

1. 打开 AdvancedClock
2. 创建或编辑闹钟
3. 在"🔊 闹钟声音"区域点击"浏览..."
4. 选择本目录中的声音文件
5. 点击"试听"测试效果
6. 点击"确定"保存
