# 自定义闹钟声音目录

## 📁 目录说明

这个目录用于存放用户自定义的闹钟声音文件。

## 🎵 关于自定义声音

你可以将任何喜欢的音频文件放在这个目录中，作为闹钟声音使用。

## 🔧 支持的文件格式

- ✅ WAV - 无损音频格式（推荐）
- ✅ MP3 - 压缩音频格式
- ✅ WMA - Windows Media Audio
- ✅ M4A - MPEG-4 Audio

## 📝 文件要求

### 质量要求
- **采样率**：44.1 kHz 或 48 kHz
- **比特率**：128 kbps 以上（MP3）
- **声道**：单声道或立体声
- **时长**：建议 5-30 秒

### 音量要求
- 音量适中，不要过大或过小
- 建议使用 Audacity 标准化音量

### 命名建议
- 使用有意义的文件名
- 避免使用特殊字符
- 可以使用中文或英文

## 💡 自定义声音来源

### 1. 录制真实声音
- 使用手机或录音设备录制
- 录制真实的闹钟声音
- 录制铃铛、风铃等声音
- 录制自然声音（鸟鸣、流水等）

### 2. 使用喜欢的音乐
- 从音乐文件中截取片段
- 使用 Audacity 编辑
- **注意版权问题**

### 3. 录制语音提醒
- 录制自己的声音："该起床了！"
- 录制家人的声音
- 使用 TTS 工具生成语音

### 4. 从免费网站下载
- Pixabay：https://pixabay.com/sound-effects/
- Freesound：https://freesound.org/
- Mixkit：https://mixkit.co/free-sound-effects/

### 5. 使用在线工具生成
- SFXR：https://sfxr.me/ (8位音效生成器)
- Beepbox：https://www.beepbox.co/ (音乐创作)

## 🛠️ 音频编辑工具

### Audacity（推荐）
- **网址**：https://www.audacityteam.org/
- **功能**：剪辑、音量调整、格式转换
- **平台**：Windows, Mac, Linux
- **价格**：免费开源

### 常用编辑操作

#### 1. 剪辑音频
```
1. 打开音频文件
2. 选择需要的部分
3. 编辑 → 删除其他部分
4. 文件 → 导出
```

#### 2. 调整音量
```
1. 选择全部音频（Ctrl+A）
2. 效果 → 标准化
3. 设置最大振幅为 -1.0 dB
```

#### 3. 添加淡入淡出
```
1. 选择开头部分 → 效果 → 淡入
2. 选择结尾部分 → 效果 → 淡出
```

#### 4. 格式转换
```
1. 文件 → 导出 → 导出为 WAV/MP3
2. 选择合适的质量设置
3. 保存文件
```

## 📂 使用方法

### 添加自定义声音

1. **准备音频文件**
   - 确保格式为 WAV, MP3, WMA 或 M4A
   - 建议时长 5-30 秒
   - 音量适中

2. **放置文件**
   - 将文件复制到本目录
   - 或放在任意位置

3. **在应用中选择**
   - 打开 AdvancedClock
   - 创建或编辑闹钟
   - 点击"浏览..."按钮
   - 选择自定义声音文件
   - 点击"试听"测试效果
   - 点击"确定"保存

## 📊 文件管理建议

### 组织方式

```
custom/
├── personal/           # 个人录制
│   ├── my_voice.wav
│   └── family_voice.wav
├── music/             # 音乐片段
│   ├── morning_song.mp3
│   └── energetic_music.mp3
├── nature/            # 自然声音
│   ├── birds.wav
│   └── ocean.wav
└── effects/           # 音效
    ├── game_sound.wav
    └── alert.wav
```

### 命名建议

**好的命名**：
- `morning_wake_up.wav`
- `gentle_piano_music.mp3`
- `emergency_alert.wav`
- `家人的声音.wav`

**不好的命名**：
- `sound1.wav`
- `新建文件.mp3`
- `未命名.wav`

## 💬 常见问题

### Q: 支持哪些音频格式？
A: WAV, MP3, WMA, M4A

### Q: 音频文件应该多长？
A: 建议 5-30 秒，太短可能不够响亮，太长可能太吵

### Q: 如何调整音量？
A: 使用 Audacity 的"标准化"功能

### Q: 可以使用流行歌曲吗？
A: 仅供个人使用可以，但要注意版权问题

### Q: 文件放在其他位置可以吗？
A: 可以，应用支持选择任意位置的音频文件
