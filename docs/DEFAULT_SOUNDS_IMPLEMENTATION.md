## 🚀 用户使用流程

### 方法1：使用下载助手脚本（推荐）

1. **运行脚本**
   ```powershell
   cd E:/code/AdvancedClock/sounds
   .\download_helper.ps1
   ```

2. **选择操作**
   - 选项 1：打开 Pixabay 音效页面
   - 选项 4：打开 defaults 目录

3. **下载音效**
   - 在 Pixabay 搜索并下载喜欢的声音
   - 保存到 defaults 目录

4. **重命名文件**
   - 按照推荐格式重命名
   - 例如：`alarm_01_classic_bell.wav`

5. **在应用中使用**
   - 打开 AdvancedClock
   - 创建或编辑闹钟
   - 选择声音文件
   - 试听并保存

### 方法2：手动下载

1. **访问 Pixabay**
   - 打开 https://pixabay.com/sound-effects/search/alarm/

2. **搜索音效**
   - 搜索关键词：alarm, bell, notification

3. **下载文件**
   - 点击"免费下载"
   - 保存到 `sounds/defaults/` 目录

4. **重命名文件**
   - 使用推荐的命名格式

5. **在应用中使用**
   - 按照正常流程选择声音文件

## 🔧 音频文件要求

### 格式要求
- **支持格式**：WAV, MP3, WMA, M4A
- **推荐格式**：WAV（无损）或 MP3（压缩）

### 质量要求
- **采样率**：44.1 kHz 或 48 kHz
- **比特率**：128 kbps 以上（MP3）
- **声道**：单声道或立体声
- **时长**：5-30 秒

### 音量要求
- **音量**：适中，不要过大或过小
- **标准化**：建议使用 Audacity 标准化音量

### 命名规范
- **格式**：`alarm_[编号]_[描述].wav`
- **示例**：`alarm_01_classic_bell.wav`

## 🛠️ 音频编辑工具

### Audacity（推荐）
- **网址**：https://www.audacityteam.org/
- **功能**：剪辑、音量调整、格式转换
- **平台**：Windows, Mac, Linux
- **价格**：免费开源

### 常用操作
1. **剪辑音频**：选择需要的部分，删除其他
2. **调整音量**：效果 → 标准化
3. **添加淡入淡出**：效果 → 淡入/淡出
4. **格式转换**：文件 → 导出
