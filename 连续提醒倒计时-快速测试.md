# 连续提醒倒计时 - 快速测试指南

## 🚀 30秒快速测试

### 创建测试闹钟

1. **点击"添加闹钟"按钮**

2. **填写以下信息**：
   ```
   名称：连续提醒测试
   时间：30秒后
   循环模式：不循环
   启用提前提醒：✅
   提前时间：20秒
   重复间隔：5秒
   ```

3. **点击"确定"**

### 观察倒计时变化

#### 阶段1：准备阶段（0-10秒）
- **提前提醒倒计时列**显示：`20秒` → `19秒` → ... → `10秒`
- **颜色**：绿色 → 琥珀色
- **说明**：距离首次提醒还有20秒倒数到10秒

#### 阶段2：首次提醒（10秒时）
- **系统提示**：弹出提醒通知
- **倒计时立即切换**：显示 `5秒`（蓝色）
- **说明**：首次提醒触发，现在显示距离第2次提醒的倒计时

#### 阶段3：提醒期间（10-25秒）
每隔5秒重复以下循环：
```
提醒触发时刻：
├─ 显示：5秒（蓝色）- 刚提醒过
├─ 3秒后：2秒（琥珀色）- 中期
├─ 4.5秒后：0.5秒（深橙色）- 即将提醒
└─ 5秒后：提醒！重新开始循环

时间点：
- 10秒：首次提醒 → 显示5秒倒计时
- 15秒：第2次提醒 → 显示5秒倒计时
- 20秒：第3次提醒 → 显示5秒倒计时
- 25秒：第4次提醒 → 进入最后阶段
```

#### 阶段4：最后阶段（25-30秒）
- **显示内容**：`最后 5秒` → `最后 4秒` → ... → `最后 1秒` → `即将触发`
- **颜色**：紫色 → 深粉色（最后30秒以内）
- **说明**：下一次间隔提醒（30秒）已超过闹钟时间（30秒），显示距离闹钟的倒计时

#### 阶段5：闹钟触发（30秒时）
- **系统弹窗**：闹钟触发提示
- **倒计时**：显示"已过期"

## ✅ 验证要点

测试时请确认以下功能正常：

- [ ] **倒计时连续性**：每秒更新，无跳跃或卡顿
- [ ] **颜色动态变化**：蓝色→琥珀色→橙色→深橙色平滑过渡
- [ ] **提醒触发准时**：每5秒准时弹出提醒
- [ ] **倒计时重置**：每次提醒后倒计时重新开始
- [ ] **最后阶段标识**：显示"最后 X秒"前缀
- [ ] **紫色/深粉色**：最后阶段颜色正确

## 🎨 颜色观察重点

### 提醒期间颜色循环（每5秒一轮）
```
0.0秒 提醒触发！ ────┐
0.0-2.5秒 蓝色       ├─ 刚提醒过，时间充足
2.5-3.5秒 琥珀色     ├─ 进入中期
3.5-4.5秒 橙色       ├─ 临近下次提醒
4.5-5.0秒 深橙色     ├─ 即将再次提醒
5.0秒 提醒触发！ ────┘ 循环
```

### 最后阶段颜色（25-30秒）
```
25.0-29.7秒 紫色     ─ 最后阶段
29.7-30.0秒 深粉色   ─ 最后30秒
```

## 💡 注意事项

1. **时间精度**：倒计时精确到秒，但提醒可能有100-200毫秒误差
2. **系统负载**：如果系统繁忙，颜色切换可能略有延迟
3. **窗口焦点**：测试时保持程序窗口可见以便观察
4. **音效测试**：确保系统音量开启以听到提醒音效
