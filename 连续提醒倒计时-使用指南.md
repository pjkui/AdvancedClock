# 连续提醒倒计时 - 使用指南

## 📖 关于本功能

**连续提醒倒计时**是高级闹钟程序的核心功能之一，它能够在提醒期间持续显示距离下一次间隔提醒的倒计时，让您始终清楚下一次提醒何时到来。

## 🎯 一句话说明

> 提前提醒倒计时列会自动检测后续间隔提醒，并实时显示距离下次提醒的倒计时，确保连续提醒的及时性。

## 🚀 快速开始

### 步骤1：创建闹钟

1. 点击"添加闹钟"按钮
2. 填写闹钟基本信息（名称、时间等）
3. ✅ 勾选"启用提前提醒"
4. 设置"提前时间"（如：10分钟）
5. 设置"重复间隔"（如：2分钟）
6. 点击"确定"

### 步骤2：观察倒计时

在"提前提醒倒计时"列中观察：

**阶段1 - 准备阶段**（绿色）
```
显示：10:00 → 9:59 → ... → 0:01
说明：距离首次提醒还有XX分XX秒
```

**阶段2 - 提醒期间**（蓝色→橙色循环）
```
首次提醒触发！
显示：2:00 → 1:59 → ... → 0:01 → 提醒！
显示：2:00 → 1:59 → ... → 0:01 → 提醒！
说明：每2分钟重复一次，显示距离下次提醒的倒计时
```

**阶段3 - 最后阶段**（紫色）
```
显示：最后 2:00 → 最后 1:59 → ... → 即将触发
说明：不会再有间隔提醒，显示距离闹钟的倒计时
```

### 步骤3：理解颜色

- 🟢 **绿色/蓝色** - 时间充足，继续当前工作
- 🟡 **琥珀色** - 中期，开始关注时间
- 🟠 **橙色** - 临近，准备收尾
- 🔴 **深橙色** - 紧急，即将提醒
- 🟣 **紫色/深粉** - 最后阶段，准备行动

## ⚙️ 参数调整建议

### 提前时间设置

| 任务类型 | 建议提前时间 | 理由 |
|---------|------------|------|
| 紧急会议 | 15-30分钟 | 需要充分准备时间 |
| 日常任务 | 5-10分钟 | 足够完成当前工作 |
| 简单提醒 | 2-5分钟 | 快速响应即可 |
| 长期规划 | 1天或更长 | 提前做好安排 |

### 重复间隔设置

| 任务类型 | 建议重复间隔 | 理由 |
|---------|------------|------|
| 紧急任务 | 2-3分钟 | 保持高度警觉 |
| 一般任务 | 1分钟 | 平衡提醒频率 |
| 轻度提醒 | 5-10分钟 | 避免过度打扰 |

### 组合策略

**策略1：渐进式提醒**
```
提前时间：长
重复间隔：短
效果：频繁提醒，逐步增强紧迫感
适用：重要会议、考试、航班
```

**策略2：轻量级提醒**
```
提前时间：短
重复间隔：长
效果：不打扰正常工作，关键时刻提醒
适用：日常任务、习惯养成
```

**策略3：密集提醒**
```
提前时间：中
重复间隔：短
效果：持续保持注意力
适用：番茄工作法、限时任务
```

## 🔧 常见调整

### 情况1：提醒太频繁

**症状**：感觉被打扰，影响专注

**解决**：
1. 增加重复间隔（如：1分钟 → 3分钟）
2. 减少提前时间（如：15分钟 → 5分钟）
3. 考虑是否真的需要提前提醒

### 情况2：提醒太少

**症状**：容易忘记，反应不及时

**解决**：
1. 减少重复间隔（如：5分钟 → 2分钟）
2. 增加提前时间（如：5分钟 → 10分钟）
3. 启用强提醒功能

### 情况3：最后阶段太短

**症状**：最后阶段感觉仓促

**解决**：
1. 增加提前时间
2. 调整重复间隔，确保最后阶段足够长
3. 计算公式：最后阶段 ≈ 重复间隔时长

### 情况4：准备阶段太长

**症状**：前期关注度不够

**解决**：
1. 减少提前时间
2. 或者接受绿色时期，专注当前工作
